# Engine Classes

## Core Classes
- **RWorldPort** - Main game world container
- **RScenePort** - Scene/location manager
- **RAnimation** - Sprite animation controller
- **RCharacter** - Character with animations and dialogue
- **RBackPack** - Inventory system
- **RLapTrap** - LapTrap device (hint system)
- **RQueue** - Event/action queue
- **RText** - Text display and dialogue
- **RHotSpot** - Clickable interaction zones
- **RSmackerMovie** - Smacker video player
- **RCompositeAction** - Multi-step action sequences
